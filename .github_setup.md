# GitHub 仓库设置指南

## 创建 GitHub 仓库

1. 登录 GitHub，点击右上角的 "+" 号，选择 "New repository"

2. 填写仓库信息：
   - **Repository name**: `mindvoice`
   - **Description**: `AI-powered cross-platform desktop voice assistant - Speech-to-text, voice notes, and translation`
   - **Visibility**: Public (开源项目)
   - **不要**勾选 "Initialize this repository with a README"（因为我们已经有了）

3. 点击 "Create repository"

## 初始化本地 Git 仓库并推送

在项目根目录执行以下命令：

```bash
# 初始化 Git 仓库（如果还没有）
git init

# 添加所有文件
git add .

# 提交更改
git commit -m "Initial commit: MindVoice - AI-powered voice assistant"

# 添加远程仓库（将 YOUR_USERNAME 替换为你的 GitHub 用户名）
git remote add origin https://github.com/YOUR_USERNAME/mindvoice.git

# 推送到 GitHub
git branch -M main
git push -u origin main
```

## 重要提醒

⚠️ **在推送之前，请确保以下文件不会被提交**：

1. **检查 `.gitignore` 文件**，确保包含：
   - `config.yml` (包含 ASR 令牌)
   - `config.json`
   - `.env`
   - `venv/`
   - `node_modules/`
   - `*.log`

2. **验证敏感信息未被提交**：
```bash
# 检查是否有敏感文件被跟踪
git ls-files | grep -E "(config\.yml|config\.json|\.env)"

# 如果看到 config.yml 等文件，需要从 Git 中移除：
git rm --cached config.yml
```

3. **检查提交历史**：
```bash
# 查看将要提交的文件
git status

# 查看提交内容（确保没有敏感信息）
git diff --cached
```

## 设置 GitHub Pages（可选）

如果需要文档网站：

1. 进入仓库 Settings
2. 选择 Pages
3. Source 选择 `main` 分支，`/docs` 目录
4. 保存

## 创建 Release

1. 进入仓库的 Releases 页面
2. 点击 "Create a new release"
3. 填写版本号（如 `v1.0.0`）
4. 添加发布说明
5. 发布
