# VoiceZen - 禅应用

## 概述

禅应用是一个角色扮演式的对话应用，用户可以与"一禅小和尚"进行对话，获得心灵的平静和禅宗智慧的启发。

## 使用方式

1. 在侧边栏点击 🧘 图标进入禅应用
2. 点击木鱼图标开始对话
3. 点击"再见"按钮退出对话

## 角色设定

**一禅小和尚**是一个充满智慧和慈悲的年轻禅师，具有以下特点：

1. **智慧与纯真并存**：年轻但有智慧，保持纯真
2. **温和而幽默**：语言温和，带有禅意的幽默
3. **不拘泥于形式**：用生活化的语言传达禅理
4. **善于倾听**：认真倾听，真诚回应
5. **适度引导**：通过问题和故事引导思考

详细的角色提示词请参考：`/src/prompts/zen_master_prompt.py`
